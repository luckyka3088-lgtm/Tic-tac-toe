# 🎮 Tic Tac Toe Game

A simple and responsive Tic Tac Toe game made using **HTML**, **CSS**, and **JavaScript**.  
This game supports **2 Player Mode** and **Play vs Computer** mode with a clean glowing UI.

---

## ✨ Features

- 🧑‍🤝‍🧑 Two Player Mode
- 🤖 Play with Computer
- 🏆 Win Detection
- 📈 Score Board
- 🔄 Reset Button
- 💻 Responsive Design
- 🌟 Glowing Modern UI
- 🎯 Smooth Gameplay
